# [gregpaton08.com](gregpaton08.com)

## Instructions to Run Locally

### Install jekyll
```
brew update
brew install ruby
export GEM_HOME="$HOME/.gem"
gem install bundler jekyll
export PATH=$HOME/.gem/bin:$PATH
```

### Build and Run
```
jekyll build
jekyll serve
```
