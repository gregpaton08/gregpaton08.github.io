Hardware

The hardware is a slightly modified Westinghouse 28068 Indoor 2-outlet Remote that I picked up for cheap on a post-Christmas sale. The unit consists of a box that sits between the lamp's power cord and the wallet outlet along with a remote to control it. The portion that plugs into the wall was left as is. The remote was ripped apart, a few resistors and transistors were added, and now it's controlled using a Raspberry Pi which exposes control via a REST API and a basic web page that I can access on my local network.
